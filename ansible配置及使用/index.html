ansible配置及使用

ansible安装

yum install ansible

ansible 版本

配置无密码访问

  1. 生成密钥
    ssh-keygen
  2. 密钥下发
    ssh-copy-id [-h|-?|-n] [-i [identity_file]] [-p port] [[-o <ssh -o options>]...] [user@]hostname
    或手动拷贝被下发服务器，执行
    cat id_rsa.pub >> /root/.ssh/authorized_keys
  3. 测试无密钥访问
    ssh [user@]hostname

添加主机并测试访问

  1. 编辑/etc/ansible/hosts,添加

  2. 使用ping测试连通

playbook使用

test.yaml

  • hosts: 目标主机
  • remote_user: 远程ssh认证用户
  • gather_facts: 设置’facts’信息收集
  • tasks: 任务
  • name: 任务名称

  • shell： 执行shell模块

    执行方式: ansible-playbook test.yaml
    ansible支持的模块: ansible-doc -l
    ansible指定模块参数查看: ansible-doc -s shell
