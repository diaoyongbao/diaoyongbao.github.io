hexo安装使用

  1. 切换淘宝源
    npm config set registry https://registry.npm.taobao.org

  2. 安装使用hexo-cil

  3. 安装部署到github.io的依赖
    npm install –save hexo-deployer-git

  4. 修改配置文件部署到github.io

  5. 部署项目,访问项目地址
    hexo deploy

    next安装使用

  6. 安装

  7. 修改项目的配置文件启用
    theme: next
